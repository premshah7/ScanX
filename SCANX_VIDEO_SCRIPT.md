# ScanX 60-Second Video Script

## Title
**ScanX — Smart, Secure Attendance for Modern Campuses**

## 0–5s | Intro
- Visual: Logo/title card with subtle motion.
- Voiceover: “Meet ScanX — a secure attendance platform built for modern campuses.”
- On-screen text: “ScanX — Smart. Secure. Real-time.”

## 5–15s | Problem Statement
- Visual: Text overlay or simple animated problem cards.
- Voiceover: “Manual attendance is slow and easy to cheat. ScanX replaces paper and proxy risk with QR-based check-in and device verification.”
- On-screen text: “Problem: slow attendance, proxy fraud, weak security.”

## 15–25s | Student Experience
- Visual: Mobile screenshot with “Mark Attendance” and scanner UI.
- Voiceover: “Students scan the session QR, and ScanX verifies their device instantly.”
- On-screen text: “Scan QR → Verify device → Mark attendance.”

## 25–35s | Faculty Experience
- Visual: Faculty dashboard screenshot with session start and attendance metrics.
- Voiceover: “Faculty start sessions in one click, generate secure QR codes, and monitor attendance live.”
- On-screen text: “Start session → Publish QR → Track attendance live.”

## 35–45s | Security & Anti-Proxy
- Visual: Proxy alert or live activity feed screenshot.
- Voiceover: “Proxy attempts are detected immediately. Unauthorized check-ins are logged so admins can act fast.”
- On-screen text: “Device binding • Proxy detection • Live alerts.”

## 45–55s | Admin Command Center
- Visual: Admin dashboard screenshot with analytics cards.
- Voiceover: “Admins see campus-wide attendance trends, security metrics, and full control from one command center.”
- On-screen text: “Campus insights • Security metrics • Central control.”

## 55–60s | Closing
- Visual: Final call-to-action card with GitHub/demo link.
- Voiceover: “ScanX — smarter, safer attendance. Check it out on GitHub.”
- On-screen text: “ScanX on GitHub | Let’s connect.”

## Notes
- Keep the pace tight and confident.
- Use screenshot transitions and clean text overlays.
- Focus on product value: speed, security, reliability.
